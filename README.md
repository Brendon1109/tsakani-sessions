# Tsakani Sessions Website

A modern, responsive website for Tsakani Sessions - a music event and content creation service.

## Features

- **Responsive Design**: Optimized for all devices
- **Booking System**: Interactive forms for different service packages
- **Social Media Integration**: YouTube videos and Instagram photos
- **Contact Information**: Phone, email, and social media links
- **Newsletter Subscription**: Email signup for event updates
- **WhatsApp Integration**: Direct link to WhatsApp channel

## Services Offered

1. **Full Tsakani Session**
   - Content creation and camera crew
   - Multiple DJs
   - Setup assistance and crowd control
   - DJ equipment provided

2. **DJ Services Only**
   - Professional DJ selection
   - Customizable playlists
   - Event-specific music curation

3. **Content Creation & Camera Crew**
   - Professional videography
   - Social media content
   - Event photography
   - Post-production included

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Inter)

## Setup

1. Clone the repository
2. Open `index.html` in a web browser
3. No build process required - it's a static website
